you launch copy for the first wave of GitHub stars.

## Positioning

Primary angle:

```text
FitSnap Coach is a browser-only AI fitness coach MVP that combines meal logging,
training plans, MoveNet form checks, recovery scoring, and an agent action queue
without a backend.
```

Why it is worth sharing:

- It is an inspectable AI product prototype, not just a landing page.
- It demonstrates local-first data handling for a health-tech workflow.
- It combines nutrition, training, form, recovery, and agent UX in one small app.
- It is easy to run because it is a static site.

## Launch Checklist

- [ ] Add a license before asking people to reuse the code.
- [ ] Upload `docs/assets/fitsnap-dashboard.png` or a cropped 1280x640 variant
      as the GitHub social preview image.
- [ ] Pin the repository on the GitHub profile.
- [ ] Add the live demo URL to the repository homepage field.
- [ ] Create a new release after the README update.
- [ ] Post to 3 focused communities instead of many broad channels.
- [ ] Reply to every first-week issue or comment within 24 hours.
